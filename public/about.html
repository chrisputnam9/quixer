Quixer is a Quick Search tool. It combines features inspired by Custom Search Engines, DuckDuckGo's "Bangs", and other tools. Quixer adds syncing of customizations across browsers & devices.

DDG's Bang's give a great way to access a large number of useful services with an easy shortcut - saving a few clicks or keystrokes.

Custom Search Engines provide a way for you to create your own shortcuts for similar purposes. You can use them to create bookmark aliases for yourself, search shortcuts for sites you use a lot, quickly compose e-mails, look up IP addresses - just about anything you can think of that can fit into a URL with a dynamic component.

Quixer aims to combine these two concepts, and adds the ability to sync those customizations across any browser and device you use.

The current release provides:

  • An interface to manage and sync up your services (search URLs)
  • A selection of default services for popular sites
  • Ability to integrate with your browser's search bar (via a custom search engine - the app supports queries via URL)
  • Ability to sync your custom services/URLs across browsers/devices via Google Drive

‚ö° Get Started

  1. Go to https://quixer.onl/#config, customize the services you want to use, and set up syncing with Google Drive (optional).
  2. Set up https://quixer.onl?q=%s as your default custom search engine (eg. in Chrome - chrome://settings/searchEngines)
  3. Start searching, for example, try "map nearest restaurant" in your search bar
