What is a custom search engine?

Quixer App | About Quixer | All Tips

A custom search engine is a keyword shortcut for a URL or command - not necessarily a search - set in your browser or another tool.

An Example

Say you want to search Etsy for a tea cozy. Normally, you would first go to etsy.com, then type in "tea cozy", click search / hit enter, and then you'd be off and shopping. This one is fun!

Now, suppose you shop on Etsy a lot - you might set up a custom search engine for yourself to save a few steps. Once it's set up, you might now simply type into your browser search bar 'etsy tea cozy' and boom, you're off!
