Custom Search Engine Hacks

Quixer App | About Quixer | All Tips

Another way to think of a custom search engine: it is a URL with a "slot" in it that is filled in with the text you enter in your URL bar.

Often the text you enter would be a search phrase, and the URL would go to a search results page - but that doesn't have to be the case.

Here are some ideas to get you started!

Simple Keyword Shortcuts - Bookmark Alternative
Keyword URL
cal https://calendar.google.com/calendar/r/month
drafts https://mail.google.com/mail/u/0/#search/in:draft
Searches With a Bit Extra
Keyword URL
fromhome https://www.google.com/maps/dir/home/%s
events https://calendar.google.com/calendar/u/0/r/search?q=%s
Information Lookup
Keyword URL
wttr https://wttr.in/%s
ip https://ip-api.com/#%s
caniuse https://ip-api.com/#
App Actions
Keyword URL
timer https://e.ggtimer.com/%s?theme=gg_timer_digital&hideToolbar=true&hideNudge=true
gmc https://mail.google.com/mail/?view=cm&fs=1&to=%s
event https://calendar.google.com/calendar/render?action=TEMPLATE&text=%s
event https://docs.google.com/document/create?title=%s
