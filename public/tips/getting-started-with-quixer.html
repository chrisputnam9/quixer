Getting Started With Quixer

Quixer App | About Quixer | All Tips

  1. Set up https://quixer.onl?q=%s as your default custom search engine in your Browser (eg. in Chrome - chrome://settings/searchEngines)
  2. Start searching, for example, try "map nearest restaurant" in your search bar
  3. Go to https://quixer.onl/#config, customize the services you want to use, and if you like, set up syncing with Google Drive.
  4. Read about custom search engine hacks to inspire adding more to your Quixer configuration.
